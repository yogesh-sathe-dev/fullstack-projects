## 🏆 Featured Project: Job Application Tracker

A full stack MERN application to track job applications with powerful features:

- 📊 **Dashboard** — Stats, charts, and insights
- 📋 **Kanban Board** — Drag & drop to manage application stages
- 🔐 **Authentication** — JWT with refresh tokens
- 📝 **Notes & Contacts** — Keep track of every interaction
- 📅 **Interview Scheduler** — Never miss an interview
- 📤 **Resume Upload** — Cloudinary integration
- 🌙 **Dark Mode** — Full theme support
- ✅ **Tests** — Unit & integration tests with Vitest
- 🔄 **CI/CD** — GitHub Actions for automated testing

**Tech Stack:** React, TypeScript, Node.js, Express, MongoDB, Zustand, Tailwind CSS, Cloudinary, Vitest, GitHub Actions
